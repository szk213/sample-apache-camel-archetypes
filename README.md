# sample-apache-camel-archetypes

## api-component

```sh
mvn archetype:generate -DarchetypeGroupId=org.apache.camel.archetypes \
    -DarchetypeArtifactId=camel-archetype-api-component \
    -DarchetypeVersion=3.0.0 \
    -DgroupId=com.szk213.samples \
    -DartifactId=camel-archetype-api-component \
    -Dname=Hello \
    -Dscheme=hello \
    -Dversion=1.0-SNAPSHOT \
    -DinteractiveMode=false
```
